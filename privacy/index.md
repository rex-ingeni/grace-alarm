# 개인정보 처리방침 — 찬송가 알람 (Chant Alarm)

최종 갱신: 2026-05-31 · 버전 1.0

## 한 줄 요약
찬송가 알람은 로그인·계정이 없으며, 알람과 설정을 **기기 안에만** 저장합니다. 외부와의 통신은 **찬송가 음원 다운로드**와 **광고 표시(Google AdMob)** 두 가지뿐입니다.

## 1. 수집·이용하는 정보
- 앱 자체는 이름·이메일·전화번호·위치·연락처·사진·캘린더 등 개인정보를 **수집하지 않습니다.** 계정·로그인 기능이 없습니다.
- 다만 광고 표시를 위해 **Google AdMob SDK**가 광고 식별자와 기기·네트워크 정보를 수집·이용할 수 있습니다(맞춤형 또는 비맞춤형 광고 제공 목적).
  - 유럽 경제 지역(EEA) 등 적용 지역에서는 **동의 관리 플랫폼(UMP)**을 통해 광고 개인화 동의를 먼저 요청합니다.
  - AdMob의 데이터 처리에 관한 자세한 내용: <https://policies.google.com/privacy>
- **광고 제거 인앱결제**: 결제는 **Google Play**가 처리하며, 앱은 카드·결제 정보를 수집·저장하지 않습니다. 앱은 '광고 제거' 보유 여부만 기기에 로컬로 기록합니다.

## 2. 데이터 저장 위치
- 알람·테마 등 모든 설정은 **기기 내부 저장소(SharedPreferences)**에만 저장됩니다. 자체 서버나 클라우드로 전송·백업하지 않습니다.
- 다운로드한 찬송가 음원은 앱 전용 저장 공간에 보관됩니다.

## 3. 외부 통신
앱이 네트워크와 통신하는 경우는 다음 두 가지뿐입니다.
1. **찬송가 음원 다운로드** — GitHub Pages 다운로드 저장소에서 선택한 곡 파일을 내려받습니다.
2. **광고 표시** — Google AdMob 서버에 광고를 요청·표시합니다(메인 알람 목록 화면 하단 배너).

알람이 울리는 화면, 알람 편집 화면, 찬송가 선택 화면에는 광고를 표시하지 않습니다.

## 4. 권한 요청
앱이 요청하는 Android 권한과 용도:
- **인터넷(INTERNET)** — 찬송가 다운로드, 광고 표시
- **정확한 알람(SCHEDULE_EXACT_ALARM / USE_EXACT_ALARM)** — 지정 시각 알람 발생
- **부팅 완료 수신(RECEIVE_BOOT_COMPLETED)** — 재부팅 후 알람 재예약
- **전체 화면 알림(USE_FULL_SCREEN_INTENT)** — 잠금화면 위 알람 화면 표시
- **진동(VIBRATE)**, **절전 해제(WAKE_LOCK)** — 알람 울림
- **알림(POST_NOTIFICATIONS)** — 알람 알림 표시
- 광고 SDK가 광고 식별자 접근 권한(AD_ID)을 자동으로 추가할 수 있습니다.

## 5. 데이터 삭제
- 앱을 삭제하면 기기에 저장된 알람·설정·다운로드 음원이 모두 함께 삭제됩니다.
- 광고 개인화를 초기화하려면 기기 설정에서 광고 ID를 재설정할 수 있습니다.

## 6. 만 14세 미만 아동
이 앱은 만 14세 미만 아동을 대상으로 하지 않는 **일반 사용자용** 앱입니다. 아동의 개인정보를 의도적으로 수집하지 않습니다.
